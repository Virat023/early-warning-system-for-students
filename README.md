# Early Warning System for Identifying At-Risk Students

## Project Overview
This project develops a machine learning based Early Warning System (EWS) that identifies students at risk of poor academic performance or dropout using student academic and attendance data.

## Objectives
- Identify at-risk students early
- Support educational planning
- Provide timely intervention for student improvement

## Technologies Used
- Python
- Machine Learning
- Random Forest
- Logistic Regression
- Decision Tree
- Flask (Web Interface)
- Pandas
- Scikit-Learn

## Dataset
Student dataset including:
- Attendance
- Math score
- Reading score
- Writing score

Dataset Sources:
- Kaggle
- GitHub repositories

## Algorithms Used
- Random Forest
- Logistic Regression
- Decision Tree

## System Architecture
1. Data Collection
2. Data Preprocessing
3. Model Training
4. Prediction System
5. Web Interface

## Results
The model predicts whether a student is:
- At Risk
- Not At Risk

The system provides quick predictions through a simple user interface.

## Future Improvements
- Add larger datasets
- Improve prediction accuracy
- Add real-time monitoring system
- Integrate with institutional databases
